Description détaillée

Représente un noeud d'action dans le grafcet Les noeuds d'action n'exécute qu'une seule fois leur opération, à l'activation (front montant) car on assume que les actions réalisées dans le noeud sont non bloquantes. Une action bloquante dans le noeud empêchera le graphe d'exécuter les noeuds voisins.

Si synchronize == true, le noeud attend que toutes ses transitions enfants soient validée avant de transitionner.

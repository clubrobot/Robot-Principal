Description détaillée

Wrappers required to use newlib malloc-family within FreeRTOS.

Overview
Route FreeRTOS memory management functions to newlib's malloc family. Thus newlib and FreeRTOS share memory-management routines and memory pool, and all newlib's internal memory-management requirements are supported.
